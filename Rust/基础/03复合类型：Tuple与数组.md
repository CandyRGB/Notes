# <center>复合类型</center>

## Tuple

### 创建
```rust
let t: (i8, u32, f64) = (9, 15, 6.3);   //tuple中的元素类型不必相同
```

### 获取Tuple的元素值
```rust
let (x, y ,z) = t;              //使用模式匹配解构
```

### 访问Tuple元素
```rust
println!("{},{},{}", t.0, t.1, t.2);
println!("{},{},{}", x, y, z);
```

## 数组

### 声明
数组的类型用[类型;长度]表示。例如：
```rust
let arr: [i32; 5] = [1, 2, 3, 4, 5]
```
如果数组内元素相同，那么可以用以下声明方法：
```rust
let arr = [2; 5]    //arr = [2, 2, 2, 2, 2]
```
